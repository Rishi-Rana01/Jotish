# 🌌 Jotish

**Jotish** is a premium, feature-rich astrology application designed with a stunning celestial aesthetic. It offers users a seamless way to explore astrologers, analyze photo results, and interact with cosmic data through a modern, responsive interface.

## ✨ Key Features

- **🔭 Cosmic Dashboard**: A central hub for all astrological insights with a beautifully crafted UI.
- **📜 Astrologer Explorer**: Browse and filter a list of expert astrologers with real-time availability and ratings.
- **💎 Detailed Profiles**: In-depth astrologer profiles featuring skills, experience, and personalized consultation options.
- **📸 Photo Analysis**: Specialized result pages for analyzing captured photos through an astrological lens.
- **🗺️ Celestial Mapping**: Interactive map integration using Leaflet to visualize astrologer locations.
- **📊 Real-time Analytics**: Visual representation of cosmic trends and data using Recharts.
- **📱 Fluid Responsiveness**: A fully adaptive design optimized for mobile, tablet, and desktop experiences.

## 🚀 Getting Started

### Prerequisites

- Node.js (Latest LTS recommended)
- npm or yarn

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Rishi-Rana01/Jotish.git
   cd Jotish
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```
